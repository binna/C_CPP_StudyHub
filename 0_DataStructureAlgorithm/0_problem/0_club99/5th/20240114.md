## 오늘의 학습 키워드
문자열 - [문제 보러가기](https://www.acmicpc.net/problem/11719)
  
<br>

## 제출한 코드
```
#include <iostream>
#include <string>
using namespace std;

int main(void)
{
	string str;

	while (getline(cin, str))
	{
		cout << str << endl;
	}

	return 0;
}
```
