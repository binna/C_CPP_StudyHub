** C언어는 프로그래밍 언어

** 컴파일러(Compiler)
	프로그래밍 언어로 작성된 프로그램을 컴퓨터가 이해할 수 있는 기계어로 번역하는 역할
	이렇게 번역하는 일 자체를 가리켜 '컴파일(compile)'이라 함

** 프로그램의 완성 과정
	프로그램 작성 -> 컴파일(compile) -> 링크 -> 실행파일 생성

** C언어로 프로그램을 작성한다는 것은
   함수(function)를 만들고, 만든 함수들의 실행 순서를 결정하는 것

** 기본적으로 제공되는 함수를 가리켜 '표준 함수'라 하고,
   표준 함수들의 모임을 가리켜 '표준 라이브러리'라 함
   ㄴ 표준 라이브러리에는 다양한 표준함수가 존재하기 때문에, 헤더파일도 다양하게 존재

** main 함수에서 0 반환
	이 값은 main 함수를 호출한 영역으로 전달
	main 함수는 프로그램이 시작되면 자동으로 호출되는 함수
	호출의 주체는 Windows나 Linux와 같은 운영체제 > 0은 운영체제에 전달
	그 값은 프로그램의 종료 상태를 알리는 용도 > 정상적인 종료시
	반면, 비정상적인 상황으로 인하여 종료될 때까지 일반적으로 0이 아닌 값을 전달

** C언어의 연산자는 C언어에서 제공하는 다양한 연산을 위해 약속된 기호
	ㄴ C언어를 이용해서 특정연산을 요구할 때 사용되는 약속된 기호를 가리켜 '연산자(operator)'라고 함

** 값을 저장할 수 있는 메모리 공간에 붙은 이름, 혹은 메모리 공간 자체를 가리켜 변수라 함
	ㄴ 선언된 변수에 처음 값을 저장하는 것을 가리켜 초기화라 함
	ㄴ 초기화 이후에 저장된 값을 변경할 때는 대입 또는 대입연산을 진행한다고 함

** 변수를 선언만 하고 초기화하지 않으면 쓰레기 값이 저장
	ㄴ 쓰레기 값? 아무런 의미가 없는 값

** 변수의 이름을 정할 때에는 변수의 역할에 어울리는 의미 있는 이름을 지어야 함

** 두 개의 피연산자를 요구하는 연산자를 가리켜 '이항연산자(binary operator)'

** 우선순위가 동일한 두 연산자가 하나의 수식에 존재하는 경우, 어떠한 순서대로 연산하느냐 결정해 놓은 것
