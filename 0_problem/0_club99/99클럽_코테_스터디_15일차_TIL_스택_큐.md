## 제출한 코드
```
#include <vector>
using namespace std;

vector<int> solution(vector<int> arr) 
{
    vector<int> answer;

    int temp = arr[0];
    answer.push_back(temp);
    for (int i = 1; i < arr.size(); i++)
    {
        if (temp != arr[i])
        {
            temp = arr[i];
            answer.push_back(arr[i]);
        }
    }

    return answer;
}
```

## 오늘의 회고
### 문제 풀기 위한 전략
1. 0번째에 있는 값을 temp와 결과 값에 넣는다 <br>
2. 0번 제외, for문을 돈다
   temp의 값이 같으면, continue  <br>
   temp의 값이 다르면, temp 값을 변경하고 결과 값에 넣는다 <br>
